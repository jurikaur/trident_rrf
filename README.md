## Hardware
(Hardware is self sourced mainly from lecktor.com)
- LDO 300x300 frame
- MGN9 linear rails LDO
- MGN12 linear rail DHM pro(100Cr6-medium preload)
- Z-steppers - LDO-42STH40-1684L300E(1.8)
- A & B steppers - LDO-42STH48-2004MAH(0.9)
